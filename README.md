
# Health Study Analysis

Detta projekt analyserar ett hälsodata-set med fokus på blodtryck, rökvanor och sjukdomsförekomst. Analysen innehåller beskrivande statistik, visualiseringar, simuleringar, konfidensintervall och hypotesprövning.

---

## Innehåll
- Beskrivning
- Installation
- Användning
- Data
- Resultat

---

## Beskrivning
Syftet är att:
- Utföra beskrivande analys av variabler som ålder, vikt, längd, systoliskt blodtryck och kolesterol.
- Visualisera data med olika grafer.
- Simulera sjukdomsförekomst och jämföra med verkliga data.
- Beräkna konfidensintervall för medelvärde av systoliskt blodtryck.
- Testa hypotesen: *”Rökare har högre medel-blodtryck än icke-rökare”*.
